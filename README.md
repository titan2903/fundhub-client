# crowdfunding-client

## Build Setup

* Install dependencies

```sh
    npm install
```

* Serve with hot reload at localhost:3010

```sh
    npm run dev
```

* Build for production and launch server

```sh
    npm run build
```

```sh
    npm run start
```

* Generate static project

```sh
    npm run generate
```

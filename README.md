# fundhub-client

## Build Setup

* Install dependencies

```sh
    npm install
```

* Serve with hot reload at localhost:3010

```sh
    npm run dev
```

* Build for production, generate static project and launch server

```sh
    npm run build
```

```sh
    npm run generate
```

```sh
    npm run start
```
